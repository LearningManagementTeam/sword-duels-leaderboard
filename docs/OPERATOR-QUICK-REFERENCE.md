# Operator quick reference

One-page cheat sheet for the central admin team. Details: [Daily operations](DAILY-OPERATIONS.md).

## After every round publish

1. **Rounds** — enter points → **Save & publish** (not draft).
2. **Public check** — open the regional board (June/July + region).
3. **Competition map** (optional but recommended) — Admin → **Competition map** → set milestone + short caption for the home page.
4. **Tie-breaker** — if badges show **Tie breaker**, run picks under **Manage advancement picks** after the playoff.

## End of phase (June or July)

1. Publish **Round 3** for all regions.
2. Verify survivor counts on public regional pages.
3. Admin → **Advancement** → **Lock & advance** for that phase.
4. Admin → **Competition map** → set transition milestone (e.g. June → July).
5. Upload carousel photos in **Branding** if the event look changed.

## Map vs round bar

| Tool | Updates when |
|------|----------------|
| Home competition map | You save it in Admin |
| Round progress bar on `/june/luzon` etc. | You publish a round |

## Weekly branding (optional)

- **Branding** — hero logo, up to 3 home carousel photos.
- Background is automatic (gradient) — no upload.

## TV at an event

`https://YOUR-SITE/tv?phase=june&region=luzon` — fullscreen, auto-refresh.

## Preview (training only)

`/preview` uses sample data — never for live scores.
